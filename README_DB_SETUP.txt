# خطوات إنشاء قاعدة بيانات remotemysql.com تلقائياً

1. اذهب إلى https://remotemysql.com/
2. اضغط على "Create new database".
3. سيظهر لك:
   - Database name
   - Username
   - Password
   - Server
   - Port
4. انسخ هذه البيانات وضعها في ملف config.py مكان القيم الافتراضية.

---

# مثال على التعديل في config.py:
MYSQL_CONFIG = {
    'host': 'remotemysql.com',
    'user': 'اسم_المستخدم',
    'password': 'كلمة_المرور',
    'database': 'اسم_قاعدة_البيانات',
    'port': 3306
}

---

# ملاحظات:
- لا تشارك كلمة المرور مع أي شخص.
- بعد التعديل، شغل التطبيق وسيعمل من أي جهاز.
- كل من يسجل أو يحجز موقف سيظهر عند المشرف فوراً.

إذا زودتني بالبيانات (بدون كلمة المرور إذا أردت الخصوصية)، سأعدلها لك تلقائياً.
